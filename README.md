# vilela-informatica
One-page responsiva e moderna desenvolvida para a Vilela Informática, uma empresa de consultoria e serviços de TI.
Este projeto consiste em uma landing page (página única) que serve como cartão de visitas digital da Vilela Informática. O objetivo é apresentar os principais serviços, diferenciais e informações de contato da empresa de forma clara e visualmente atraente.

Tecnologias Utilizadas
- HTML5: Estrutura e marcação semântica do conteúdo.
- CSS3: Estilização e design responsivo.
